# Employee Management Dashboard

## Overview
A React.js based Employee Management Dashboard with authentication, CRUD operations, search & filter, image upload, and print functionality.

## Tech Stack
- React.js
- React Router
- Context API
- Local Storage
- CSS (Custom UI)

## Features
- Login authentication
- Add / Edit / Delete Employee
- Image upload with preview
- Search & Filter (Name, Gender, Status)
- Print all employees
- Print single employee
- Responsive & modern UI

## Steps to Run
1. Clone repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Assumptions
- Mock authentication used
- LocalStorage used instead of backend API
